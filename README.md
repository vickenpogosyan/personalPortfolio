# Personal Portfolio

A sleek, minimal personal portfolio website (built from scratch): https://vickenpogosyan.github.io/personalPortfolio/

## Features
- Multi-page layout with easy navigation
- CSS-based animations and transitions
- Contact form with client-side handling
- Backend-less form submission using Formspree
- Custom thank-you page on successful submission

## Technologies Used
- HTML
- CSS
- JavaScript
- Formspree
